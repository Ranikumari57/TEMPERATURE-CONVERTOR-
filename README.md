## Project Title:

Temperature Converter

## Objective:

To design a digital temperature converter that measures temperature in Celsius and displays the equivalent Fahrenheit value.

## Components Required

- Temperature Sensor (LM35)
- Arduino Uno
- 16×2 LCD Display (or OLED)
- Breadboard
- Jumper Wires
- USB Cable
- Power Supply
  
## Working Principle

The LM35 sensor measures temperature in Celsius.
Arduino reads the sensor output.
Arduino converts Celsius to Fahrenheit using the formula:
Fahrenheit = (Celsius × 9/5) + 32
The LCD displays both Celsius and Fahrenheit values.

## Circuit Connections

- LM35 VCC → Arduino 5V
- LM35 GND → Arduino GND
- LM35 OUT → Arduino A0
- LCD connected to Arduino (according to LCD wiring)
  
## Applications

- Weather monitoring
- Laboratory temperature measurement
- Home automation
- Industrial temperature monitoring
